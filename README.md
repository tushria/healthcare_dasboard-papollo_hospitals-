# Healthcare Analytics Dashboard - Papollo Hospitals

## Project Overview
This project showcases an interactive **Power BI dashboard** developed to analyze patient data for **Papollo Hospitals**.  
The dashboard focuses on healthcare analytics including patient statistics, billing, and diagnosis trends.

- **Key Insights:**
  - Bed occupancy and patient count analysis
  - Billing amount comparison across different medical tests
  - Diagnosis count trends for better hospital management

- **Features:**
  - Cleaned and transformed patient data for accuracy
  - Interactive filters for Patient ID and Date Range
  - Actionable insights to support decision-making
